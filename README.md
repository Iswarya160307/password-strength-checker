\# Password Strength Checker



A simple Python command-line tool that checks the strength of a password based on length, character variety, and complexity rules.



\## Features

\- Checks for minimum length (8+ characters)

\- Checks for uppercase and lowercase letters

\- Checks for numbers and special characters

\- Gives a strength rating: Weak, Medium, or Strong

\- Provides suggestions to improve weak passwords



\## How to Run

python password\_checker.py



\## Example Output

Enter a password to check: cybersecurity



Password Strength: Weak (2/5)

Suggestions to improve:

\- Add at least one uppercase letter

\- Add at least one number

\- Add at least one special character



\## Tech Used

\- Python 3

\- Regular expressions (re module)
